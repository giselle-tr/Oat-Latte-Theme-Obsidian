#### Oat Latte

![Oat Latte](screenshot.png)

A warm, cozy Obsidian theme inspired by oat milk lattes, vintage notebooks, and quiet cafés.



Oat Latte combines soft beige, warm brown, and muted cream tones to create a comfortable writing environment without sacrificing readability. The goal is to make every note feel like writing on a lightly aged notebook page.



###### Features



* 🌾 Warm oat and coffee color palette
* 📖 Lined notebook paper background
* ✍ Elegant serif typography for long writing sessions
* 💻 Cascadia Code for inline code and code blocks
* 🖍 Soft latte-inspired text highlights
* 🌙 Carefully balanced light and dark modes
* 🤍 Minimal, distraction-free design



###### Best for



* Daily journaling
* Creative writing
* Personal knowledge management
* Long study sessions
* Literature and note-taking



###### Fonts



* Cormorant Garamond
* Libre Baskerville
* Cascadia Code



###### Installation



Download the theme from the Obsidian Community Themes browser or clone this repository into your `.obsidian/themes` folder.
